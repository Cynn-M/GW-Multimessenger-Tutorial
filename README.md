# GW-Multimessenger-Tutorial

Overview

This repository contains a collection of tutorial notebooks developed while learning gravitational-wave multimessenger astronomy and electromagnetic follow-up strategies.

The notebooks focus on:

Reading and visualizing gravitational-wave sky localization maps.
Exploring the properties of the binary neutron star event GW170817.
Learning the basic functionality of SKYFAST for rapid multimessenger analyses.
Understanding how galaxy catalog information can be combined with GW sky maps.

These notebooks were developed as part of my self-training in multimessenger astrophysics and gravitational-wave data analysis.

Scientific Motivation

Multimessenger astronomy combines gravitational-wave observations with electromagnetic information to identify and characterize astrophysical sources such as:

Binary neutron star mergers
Kilonovae
Short gamma-ray bursts

Efficient sky localization and galaxy-target ranking are essential for rapid electromagnetic follow-up observations.
