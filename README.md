# GW-Multimessenger-Tutorial

-------------------------- OVERVIEW -------------------------------------------

This repository contains a collection of my self-training tutorial notebooks developed while learning gravitational-wave multimessenger astronomy and electromagnetic follow-up strategies.

The notebooks focus on:

* Reading and visualizing gravitational-wave sky localization maps.
* Exploring the properties of the binary neutron star event GW170817.
* Learning the basic functionality of SKYFAST for rapid multimessenger analyses.
* Understanding how galaxy catalog information can be combined with GW sky maps.

-------------------------- NOTEBOOKS -----------------------------------------

01_Reading_GW170817_Skymap.ipynb

* Download and read GW sky localization maps.
* Visualize probability distributions on the sky.
* Compute credible regions.
* Explore the localization of the binary neutron star event GW170817.


SKYFAST.ipynb

* Installation and basic usage of SKYFAST.
* Rapid sky localization analyses.
* Interaction with galaxy catalogs.
* Preliminary exploration of host-galaxy identification techniques.
* Scientific Motivation
